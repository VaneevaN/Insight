## Аннотация проекта ##
Сейчас владельцы сайтов используют разные инструменты веб-аналитики, но большинство из них показывают только клики, скроллы и время на странице, а не то, куда именно смотрит пользователь и как по-разному воспринимают интерфейс разные группы аудитории. Существующие решения для отслеживания взгляда через веб-камеру обычно ограничиваются построением общей тепловой карты, а её интерпретация остаётся полностью на усмотрение владельца сайта — без структурированного сравнения по сегментам пользователей.
В проекте разрабатывается встраиваемый веб-сервис, с помощью которого владельцы сайтов могут проводить тестовые сессии с пользователями. С их явного согласия собираются данные о направлении взгляда через веб-камеру, а также краткая информация о самом участнике (возраст, пол и др.). На основе этих данных строятся тепловые карты и сравнения между группами пользователей, это позволяет увидеть, какие элементы интерфейса воспринимаются по-разному разными аудиториями.

## Цель проекта ##
Разработать веб-сервис для сбора данных о визуальном внимании пользователей веб-интерфейсов через веб-камеру и их сегментированной визуализации, позволяющий владельцам сайтов сравнивать восприятие интерфейса разными группами пользователей.

## Задачи проекта ##
1. Проанализировать существующие eye-tracking и веб-аналитические инструменты, выявить их ограничения в части сегментированного анализа.
2. Изучить и экспериментально оценить методы определения взгляда на основе компьютерного зрения в браузере.
3. Спроектировать архитектуру системы: встраиваемый SDK, backend, базу данных, личный кабинет.
4. Реализовать SDK для сбора данных о взгляде и активности с механизмом согласия и тестовых приглашений.
5. Реализовать backend и личный кабинет с визуализацией тепловых карт и сравнением метрик по сегментам пользователей.

## Планируемые результаты проекта ##
В результате проекта будет создан рабочий прототип веб-сервиса Insight: встраиваемый SDK для сбора данных о взгляде и активности пользователя с экраном согласия, backend с поддержкой нескольких подключённых сайтов и личный кабинет владельца сайта с тепловыми картами и сравнением метрик между группами пользователей. Дополнительно будет реализован экспорт собранных данных для дальнейшего статистического анализа, а также проведена экспериментальная оценка точности выбранного метода определения взгляда. Сервис будет протестирован на нескольких реальных сайтах, по итогам тестирования будут сформулированы выводы и ограничения работы.
