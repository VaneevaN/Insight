# Insight Dashboard

Личный кабинет владельца сайта — веб-приложение для управления тестовыми сессиями и просмотра аналитики.

## Что делает

- Регистрация и вход для владельцев сайтов
- Добавление сайтов и получение кода для встраивания SDK
- Создание тестовых кампаний и генерация приглашений для участников
- Просмотр тепловых карт по сессиям
- Сравнение метрик между группами (возраст, пол, профессия)

## Технологии

- React
- Recharts
