# Insight Backend

API-сервер для приёма событий от SDK, хранения данных и обслуживания личного кабинета.

## Что делает

- Регистрация и авторизация владельцев сайтов
- Управление подключёнными сайтами (выдача site_id)
- Генерация тестовых токенов для приглашения участников
- Приём событий взгляда и активности от SDK
- Агрегация данных по сессиям и сегментам
- Хранение данных в PostgreSQL

## Технологии

- Node.js + Express или FastAPI (Пока вразработке, поэтому трудно сказать)
- PostgreSQL
- JWT для авторизации
