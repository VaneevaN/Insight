# Insight SDK

Встраиваемый клиентский модуль для сбора данных о визуальном внимании пользователя

## Что делает

- Запрашивает явное согласие участника на использование веб-камеры
- Показывает анкету (возраст, пол, профессия и др.)
- Отслеживает направление взгляда через веб-камеру
- Привязывает координаты взгляда к DOM-элементам страницы
- Активируется только при наличии тестового токена в URL (?test=...) — обычные посетители сайта SDK не видят
- Отправляет собранные события в backend Insight

## Как встроить на сайт

<script src="https://cdn.insight-app.ru/sdk.js" data-site-id="ВАШ_SITE_ID"></script>
Токен для конкретной тестовой сессии передаётся через параметр URL:
https://yoursite.com/page?test=TOKEN
Токены генерируются в личном кабинете Insight.

## Технологии

- TypeScript (компилируется в один JS-файл без зависимостей)
- WebGazer.js / MediaPipe Face Mesh (Выбор будет произведен позже на осноев тестов)
