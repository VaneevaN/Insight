# Insight Backend

Backend часть проекта **Insight**
Пока что разработала с моковыми данными, в дальнейшем планируется доработка

## Возможности

- REST API для взаимодействия с клиентским приложением
- Получение и обработка данных пользовательской сессии
- Сохранение статистики
- Работа с результатами анализа внимания
- Подготовка данных для последующей визуализации

## Структура проекта

```
backend/
├── src/

│   ├── routes/
```

## Требования

- Node.js 20+
- npm 10+

## Установка

Клонировать репозиторий

```bash
git clone https://github.com/VaneevaN/Insight.git
```

Перейти в папку backend

```bash
cd backend
```

Установить зависимости

```bash
npm install
```

## Запуск

Режим разработки

```bash
npm run dev
```

Обычный запуск

```bash
npm start
```

## Архитектура

Backend построен по многослойной архитектуре:

- **Routes** — обработка HTTP-запросов
- **Controllers** — логика обработки запросов
- **Services** — бизнес-логика
- **Models** — работа с данными
- **Middleware** — промежуточная обработка запросов

Такое разделение упрощает поддержку и дальнейшее развитие проекта.

## Планируемые возможности

- Авторизация пользователей
- Хранение пользовательских сессий
- Сохранение результатов отслеживания взгляда
- Вычисление аналитических метрик
